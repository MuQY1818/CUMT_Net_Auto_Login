   # CUMT 校园网自动登录工具

   这是一个用于中国矿业大学校园网自动登录的桌面应用程序。

   ## 功能

   - 自动登录校园网
   - 支持多种运营商
   - 开机自启动选项
   - 自动登录选项

   ## 安装

   1. 克隆仓库：
      ```
      git clone https://github.com/MuQY1818/auto_login_app.git
      ```

   2. 安装依赖：
      ```
      pip install -r requirements.txt
      ```

   3. 运行程序：
      ```
      python auto_login_app.py
      ```

   ## 使用说明

   1. 输入学号和密码
   2. 选择运营商
   3. 点击登录按钮
